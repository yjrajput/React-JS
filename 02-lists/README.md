# 📋 Lists in React.js

This project demonstrates how to **render lists in React.js** using the `map()` method and how to use **unique keys** for better performance.  

---

## 🚀 Features
- Rendering lists using **`map()`** function  
- Using **unique keys** in lists  
- Handling **dynamic data (arrays/objects)**  
- Conditional rendering with lists  
- Example: **Todo List App**
